# Current features

## Plugins
- Highlight - eki-highlight, uses bold text
- Foreign - eki-foreign, uses italic green text
- Stress - eki-stress, uses all black regular weight text
- Meta - eki-meta, uses uppercase green text in small-caps format
- Sub - eki-sub, uses subscript with custom styles
- Sup - eki-sup, uses superscript with custom styles
- Eki-link - eki-link/ext-link, links to an internal word, allows for searching, if using an external link the ext-link element is used instead
- Eki-link disconnect - Adds a space to disconnect from the end of an eki-link, might not be necessary
- Eki-media - Add an image, also shown in-line in the editor
- Remove format - removes stylings
- Undo - Undo last change
- Redo - Redo last change
- Source - View direct html text
